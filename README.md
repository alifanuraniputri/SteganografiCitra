Cara Menjalankan:
- Import as Eclipse Project
- Run

Cara Menyisipkan:
- Pilih File Gambar ( di folder Gambar Uji ada gambar yang bisa dipake)
- Tulis kunci
- Tulis nama file (contoh file yang ada: "pesanalifa.txt")
- Tekan "Sisipkan ..."
- Tekan "Simpan ..."

Cara Mengekstrak:
- Pilih Gambar (yang sudah di sisipkan)
- Tulis kunci
- Tekan "Ekstrak"
